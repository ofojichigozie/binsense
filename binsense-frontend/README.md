# binsense-frontend

This folder contains the BinSense web dashboard.

## What it does

- Provides a React + Vite interface for login and the dashboard.
- Displays bin fill levels, charts, and recent readings.
- Connects to the backend API and real-time Socket.IO updates.

## Setup

1. Clone the repo and enter this folder.
   ```sh
   cd binsense/binsense-frontend
   ```
2. Install dependencies.
   ```sh
   npm install
   ```
3. Start the development server.
   ```sh
   npm run dev
   ```
4. Build for production if needed.
   ```sh
   npm run build
   npm run preview
   ```

## Useful scripts

- `npm run dev` – start the Vite dev server
- `npm run build` – create a production build
- `npm run preview` – preview the production build locally
